Frontend React del MVP.

## Responsabilidad
- Permitir cargar y editar la instancia mediante las secciones `Periodos`, `Medicos` y `Planificador`.
- Mantener un `instanceDraft` compartido en frontend.
- Consumir `POST /v1/solve`.
- Mostrar factibilidad, asignaciones, metricas y exportaciones JSON/CSV.
