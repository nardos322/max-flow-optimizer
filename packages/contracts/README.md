# packages/contracts

Contratos compartidos (`v1`) entre frontend, API y motor.

## Contenido esperado
- OpenAPI 3.1.
- JSON Schemas.
- Tipos TypeScript derivados.
- Ejemplos de request/response canonicamente validos.

## Estructura v1
```text
packages/contracts/v1/
├── openapi.yaml
└── schemas/
    ├── solve.request.schema.json
    ├── solve.response.schema.json
    ├── error.schema.json
    └── health.response.schema.json
```
